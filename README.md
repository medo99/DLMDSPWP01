# DLMDSPWP01

Python project scaffold for the IU course **DLMDSPWP01**.

## Project Structure

```text
DLMDSPWP01/
|-- data/          # Local input data (CSV files are ignored)
|-- output/        # Generated output (contents are ignored)
|-- src/           # Project source code
|-- tests/         # Pytest test suite
|-- main.py        # Application entry point
|-- requirements.txt
`-- README.md
```

## Setup

Create and activate a virtual environment, then install the dependencies:

```bash
python -m venv .venv
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Test

```bash
pytest
```
